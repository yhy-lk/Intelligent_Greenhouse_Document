# 摘要

温室设施农业通过人为调控环境条件提高作物产量和资源利用效率，但传统温室系统多采用基于阈值的开关控制方式，缺乏对环境变化的自适应能力。现有智能温室方案以单微控制器架构为主，在系统规模扩大时面临扩展性与实时性难以兼顾的问题，且较少集成大语言模型等人工智能技术，人机交互方式单一。将大语言模型与嵌入式实时控制系统结合，需要在保证分布式通信可靠性的同时满足本地控制的实时性要求。针对上述问题，本课题设计并实现了一种基于 ESP32 与 STM32 的分布式智能温室控制系统。系统采用"ESP32-S3 主节点 + 多个 STM32F407 从节点"的分布式架构，主节点负责图形界面、语音交互与大模型推理等交互层功能，从节点负责传感器采集与执行器控制等控制层功能，主从节点之间通过 CAN 总线通信。系统集成了 DeepSeek 大语言模型，实现了手动控制、自动控制和 AI 领航三种控制模式，其中自动控制采用迟滞控制策略和 PID 转速闭环算法，AI 领航模式支持语音交互驱动的智能环境分析。测试结果表明，系统传感器采样周期不超过 1 秒，PID 控制周期为 100 ms，CAN 总线波特率达 1 Mbps，各从节点可独立运行本地闭环控制，具备良好的实时性和可扩展性。

**关键词**：智能温室；分布式控制；CAN 总线；大语言模型；嵌入式系统
