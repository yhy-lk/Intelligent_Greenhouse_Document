# Abstract

Greenhouse agriculture improves crop yield and resource efficiency through controlled environmental conditions, yet conventional greenhouse systems predominantly rely on threshold-based switching control and lack the ability to adapt to environmental changes. Existing intelligent greenhouse solutions are mostly built on single-microcontroller architectures, which face trade-offs between scalability and real-time performance as the system expands, and rarely integrate artificial intelligence technologies such as large language models (LLMs), resulting in limited human-machine interaction methods. Combining LLMs with embedded real-time control systems requires maintaining distributed communication reliability while meeting local control real-time constraints.

To address these issues, this thesis designs and implements a distributed intelligent greenhouse control system based on ESP32 and STM32 microcontrollers. The system adopts a distributed architecture comprising an ESP32-S3 master node and multiple STM32F407 slave nodes. The master node handles interaction-layer functions including graphical user interface, voice interaction, and LLM inference, while slave nodes handle control-layer functions including sensor data acquisition and actuator control. Communication between master and slave nodes is carried out through a CAN bus. The system integrates the DeepSeek large language model and supports three control modes: manual control, automatic control, and AI pilot mode. The automatic control mode employs a hysteresis control strategy and a PID speed closed-loop algorithm, while the AI pilot mode enables voice-driven intelligent environmental analysis. Test results showed that the sensor sampling period did not exceed 1 second, the PID control period was 100 ms, and the CAN bus baud rate reached 1 Mbps. Each slave node operated its local closed-loop control independently, demonstrating good real-time performance and scalability.

**Keywords**: intelligent greenhouse; distributed control; CAN bus; large language model; embedded system
