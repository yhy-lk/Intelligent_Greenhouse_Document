## 硬件电路设计相关

### 1. 通风风扇驱动模块
- **驱动芯片**：TB6612 双H桥驱动模块（非分立H桥电路）
- **控制引脚**：
  - PWM 信号：TIM1 CH1 → PE9
  - 方向控制 IN1：PE10
  - 方向控制 IN2：PE11
- **编码器反馈**：保留，用于PID闭环转速控制
  - 编码器 A 相：TIM3 CH1 → PC6
  - 编码器 B 相：TIM3 CH2 → PC7
- **PID控制**：保留，离散位置式PID控制器

### 2. 水泵与加湿器驱动
- **驱动方式**：光耦隔离继电器模块（成品模块，非分立元件搭建）
- **触发电平**：支持高/低电平触发
- **控制引脚**：
  - 水泵：PD13
  - 加湿器：PE4

### 3. 供电电容配置
- **电源入口**：整个开发板的总电源入口（USB供电 5V）
- **电容配置**：
  - +5V 电源入口：10μF 电解电容 + 100nF 陶瓷电容（并联）
  - +3.3V 电源入口：10μF 电解电容 + 100nF 陶瓷电容（并联）
- **注意**：没有并联 470μF 大电容，舵机电源引脚附近也没有 470μF 电容

### 4. I2C 总线上拉电阻
- **上拉电阻**：4.7kΩ，上拉至 3.3V
- **应用引脚**：所有 I2C 通信引脚（SHT30、BH1750 共用的 I2C2 总线：PB10/PB11）

### 5. 突发大电流处理
- **当前方案**：高度依赖电源本身稳定性
- **涉及场景**：启动舵机、补光灯亮度突然拉至最大
- **未来改进**：考虑 12V 或 24V 供电方案（放在总结与展望中简要提及）
