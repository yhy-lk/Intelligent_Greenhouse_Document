# 摘要写作器

## 概述

本技能专注于论文英文摘要的撰写和审查。仅在写摘要时调用，不加载到正文写作的上下文中。

## 何时使用本技能

- 撰写英文摘要。
- 审查已有摘要的结构和时态。
- 用户要求"写摘要"、"write abstract"、"check abstract"。

## 何时不使用本技能

- 撰写论文正文。请改用 `thesis-writer`。
- 审查整篇论文。请改用 `thesis-reviewer`。

## 核心步骤

### 步骤 1：五句公式

| 句序 | 内容 | 时态 |
|---|---|---|
| 1 | 解决什么问题？背景 + 意义 | 一般现在时 |
| 2 | 为什么值得解决？现有方法的不足 | 一般现在时 |
| 3 | 挑战是什么？具体的，非方法层面的 | 一般现在时 |
| 4 | 如何解决？方法名称 + 关键组件 | 一般现在时 |
| 5 | 效果如何？具体数据/指标 | 一般过去时 |

### 步骤 2：时态规则

- 摘要中描述论文本身的内容：**一般现在时**（this paper proposes、the system achieves）
- 摘要中描述实验结果：**一般过去时**（experiments showed、results demonstrated）
- 正文章节时态（仅写摘要时参考）：
  - 绪论：现在时陈述主张，过去时引用相关工作
  - 方法论：现在时（the method does X）
  - 实验：过去时（we ran、we observed）
  - 结论：现在完成时（we have shown、we have proposed）

### 步骤 3：审查清单

1. **[检查]** 五句结构完整（不缺句）
2. **[检查]** 无空话（"comprehensive experiments"、"extensive analysis"）
3. **[检查]** 时态一致（描述用现在时，实验结果用过去时）
4. **[检查]** 句 5 包含具体数据，非泛泛而谈
5. **[检查]** 无禁用 AI 语气词
6. **[检查]** 英文语法：冠词、主谓一致、句子不超过两个从句
